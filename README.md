# 熊艺作品集 | Xiong Yi Portfolio

个人作品集网站，展示内容开发、学术写作、创意写作等方向的作品。

## 项目结构

```
├── index.html              # 首页
├── projects/               # 项目详情页
│   ├── others.html         # 其他项目（策展、文案等）
│   ├── piaotiao.html       # 纸条作文
│   ├── readingcub.html     # 阅读兽 ReadingCub
│   └── shanshuibide.html   # 山水比德
├── assets/                 # 静态资源
│   ├── css/style.css
│   ├── js/script.js
│   ├── images/             # 图片
│   └── videos/             # 视频
└── files/                  # 可下载文件
    ├── 创作作品/works_html/  # 创意写作 HTML 版
    ├── 学术论文/             # PDF 论文
    ├── 项目文案/             # 方案文档
    ├── 纸条PDF/             # 纸条作文 PDF
    └── 阅读兽EPUB/PDF/       # 阅读兽主题 PDF
```

## 部署

Push 到 `main` 分支后自动通过 GitHub Pages 部署。

## 本地预览

直接在浏览器打开 `index.html` 即可。
